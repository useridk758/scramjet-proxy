# 🚀 Scramjet Proxy Website (Python Backend)

A fully functional unblocked proxy website built with a Python Flask backend and a modern HTML/JS frontend. This project is designed to be lightweight, easy to deploy, and ready for GitHub hosting.

## ✨ Features
- Proxy Server: Python-based HTTP proxy using Flask and Requests.
- Modern UI: Clean, responsive interface with gradient background and glassmorphism effect.
- No Node.js Required: Pure Python backend—no need for Node.js or npm.
- GitHub Ready: Includes all necessary files for deployment on GitHub Pages or a VPS.
- Scramjet Integration: Built to work with the Scramjet proxy from Mercury Workshop.
